# rangka

Distribution package for the Rangka framework. Re-exports `@rangka/core` and `@rangka/shared` as a single dependency for app developers.

## Install

```bash
pnpm add rangka
```

## Usage

```typescript
import { defineModel, defineModule, boot } from 'rangka';
```

## What's included

- `@rangka/shared` — Types, builders, traits, and definitions
- `@rangka/core` — Server runtime, database sync, API generation, auth, hooks, jobs
